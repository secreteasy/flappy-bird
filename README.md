# flappy-bird
Flappy Bird


К техническим средствам предъявляются следующие требования:

-Компьютер, который поддерживает Visual Studio 2019;

-Монитор, разрешение (минимальное 1280x720, максимальное 1920x1080), диагональ от 16 дюймов;

-Клавиатура;

-Мышь.
